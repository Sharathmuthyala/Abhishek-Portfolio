# Abhishek Garakurthi - Portfolio Website

Professional portfolio website showcasing my experience as a Data & Business Intelligence Analyst.

## About

This portfolio highlights my 3+ years of experience in data visualization, business intelligence, and predictive analytics across healthcare and enterprise domains. I specialize in transforming complex datasets into actionable insights using SQL, Python, and Power BI/Tableau.

## Features

- Modern, responsive design with navy blue, white, and light gray color scheme
- Interactive navigation with smooth scrolling
- Project showcase with detailed descriptions
- Professional experience timeline
- Skills and technologies overview
- Contact information and social links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Font Awesome Icons
- Responsive Design (Mobile-First Approach)

## Setup Instructions

1. Clone this repository
2. Add your profile photo as `profile.png` in the root directory
3. Open `index.html` in your browser to view locally
4. For GitHub Pages deployment, the site will be available at: `https://sharathmuthyala.github.io/Abhishek-Portfolio/`

## GitHub Pages Deployment

This site is configured for GitHub Pages deployment. Simply push to the main branch and enable GitHub Pages in repository settings.
